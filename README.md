# Özbekistan Sağlık Sistemi - Online Randevu Sistemi

Özbekistan sağlık sistemi için geliştirilmiş modern ve kullanıcı dostu bir online randevu platformu.

## Özellikler

- **Branş Seçimi**: 10+ farklı tıbbi branş arasından seçim yapabilme
- **Doktor Seçimi**: Her branş için uzman doktor listesi
- **Gün ve Saat Seçimi**: Hafta içi günlerde 08:00-17:00 arası 15 dakikalık dilimlerle randevu alabilme
- **Dinamik Arayüz**: Kullanıcı dostu, responsive tasarım
- **Hakkımızda Bölümü**: Kurum hakkında detaylı bilgi

## Mevcut Branşlar

- Dahiliye
- Ortopedi
- Kardiyoloji
- Kadın Hastalıkları ve Doğum
- Aile Hekimliği
- Çocuk Cerrahisi
- Göğüs Cerrahisi
- Genel Cerrahi
- Göz Hastalıkları
- Nöroloji

## Teknolojiler

- **HTML5**: Sayfa yapısı
- **CSS3**: Stil ve tasarım
- **JavaScript (Vanilla)**: İnteraktif özellikler ve randevu yönetimi

## Kurulum ve Kullanım

1. Projeyi klonlayın:
```bash
git clone https://github.com/dilagencaga/health-appointment-system.git
```

2. Proje klasörüne gidin:
```bash
cd health-appointment-system
```

3. HTML dosyasını bir tarayıcıda açın:
   - `daha güzel.html` dosyasını çift tıklayarak açabilirsiniz
   - Veya bir local server kullanabilirsiniz

## Nasıl Çalışır?

1. Ana sayfada üç seçenek bulunur:
   - **Hakkımızda**: Kurum bilgilerini görüntüleme
   - **Randevu**: Randevu alma sistemi
   - **Özbekistan'da Sağlık Devrimi**: Bilgilendirme bölümü

2. Randevu alma adımları:
   - Branş seçimi yapın
   - İlgili doktoru seçin
   - Müsait günü seçin
   - Saat aralığını belirleyin
   - 15 dakikalık dilimden birini seçin
   - "Randevuyu Oluştur" butonuna tıklayın
